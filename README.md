# express-es6-boilerplate
a quick start width using es6 in express

## Usage

### start the server 
```js
npm start
```

### use nodemon to start the server 
```
npm run nstart
```

### use babel transform es6 => es5
```js
npm run build 
```

### run test
```js
npm run test
```

### run the build server
```js
npm run serve
```
